

# TriviaGame

## Installing the project
If you don't have `yarn`

Install `yarn` via `npm`
####`npm install -g yarn`

Install `Nx Nrwl Monorepo`
####`yarn global add @nrwl/cli`

Run project on port `:4200`
####`nx serve`
